# 🎓 Exam Platform Analytics Dashboard

This project is a business analytics dashboard built for an online exam platform.

It simulates real-world platform usage using synthetic data and provides insights into student activity, exam performance, revenue, and proctoring behavior.

---

## 🚀 Features

* 📊 Student & Exam Metrics
* 📉 Completion & Drop-off Analysis
* 💰 Revenue Tracking
* 🧠 Score & Pass Rate Analysis
* ⚠️ Proctoring Event Monitoring
* 📈 Daily Revenue Trends
* 🎯 Subject, Difficulty & Date Filters
* ⬇️ Downloadable Data Reports
* 🧠 Automated Business Insights

---

## 🛠️ Tech Stack

* Python
* Pandas
* Streamlit
* NumPy

---

## 📂 Project Structure

* `generate_data.py` → Generates synthetic platform data
* `app.py` → Streamlit analytics dashboard
* `data/` → Generated CSV datasets

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python generate_data.py
streamlit run app.py
```

---

## 📊 Key Insights Generated

* Completion Rate & Drop Rate
* Revenue per Quiz
* Average Score & Pass Rate
* Subject-wise Performance
* Suspicious Activity Detection (Proctoring Events)
* Daily Revenue Trends

---

## ⚠️ Note

This project uses synthetic data to simulate a real exam platform. No real user data is used.

---

## 💡 Use Case

This dashboard demonstrates how an online exam platform can be analyzed from a business and product perspective, including:

* User engagement
* Performance analytics
* Revenue insights
* Risk & cheating detection
